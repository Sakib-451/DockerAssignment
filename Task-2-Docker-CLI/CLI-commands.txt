1- docker version
 Displays the version information of the Docker Client and Docker Engine

2 - docker info   
Displays detailed information about the Docker installation and system configuration.

3 - docker images	
Lists all Docker images available on the local system.

4 - docker ps	  
Lists all currently running containers.

5 - docker ps -a	
Lists all containers, including running and stopped containers.

6 - docker pull	
Downloads a Docker image from Docker Hub.

7- docker run	  
Creates and starts a new container from a Docker image.

8 - docker exec	
Executes a command inside a running container.

9 - docker logs
Displays the logs generated by a container.

10 - docker inspect	
Displays detailed information about a Docker container in JSON format.


11 - docker stop	
Stop a running container. 

12 - docker start	
 Starts an existing stopped container.

13 - docker restart	
Restarts a running or stopped container.


14 - docker rm	
Removes one or more stopped containers

15 - docker rmi	
Removes one or more Docker images from the local system.

16 - docker stats  	
Displays live resource usage statistics for running containers, including CPU, memory, network, and disk I/O.

17 - docker system df	
 Shows the disk space used by Docker images, containers, volumes, and build cache.


18 - docker system prune	
Removes unused Docker objects such as stopped containers, unused networks, dangling images, and build cache to free up disk space.

# Docker Assignment

## Objective

The objective of this assignment is to understand Docker fundamentals including images, containers, Dockerfiles, networking, volumes, logs, and container management.

## Tasks Completed

- Task 1: Docker Installation
- Task 2: Docker CLI Commands
- Task 3: Docker Images
- Task 4: Nginx Custom Image
- Task 5: Ubuntu Custom Image
- Task 6: Docker Volumes
- Task 7: Docker Networking
- Task 8: Docker Logs
- Task 9: Docker Inspect
- Task 10: Docker Cleanup

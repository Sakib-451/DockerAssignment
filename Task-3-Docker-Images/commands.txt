1 -  Pull the image

Command

 docker pull nginx

2 - Start a container

Command

docker run -itd --name nginx-container 8080:80 nginx

3 -  Verify it is running

Command
 
Docker ps

5 - Stop the container

Command

Docker stop nginx-container


6 - Remove the container

Command

Docker rm nginx-container
